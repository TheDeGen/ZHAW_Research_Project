# Financial Sentiment Analysis for Trading Strategy Backtesting

This project aims to backtest a trading strategy based on sentiment analysis of financial news headlines from CNBC, Reuters, and The Guardian.

## Project Structure

- `data/`: Contains the raw and processed datasets.
- `notebooks/`: Jupyter notebooks for exploration and analysis.
- `src/`: Source code for data processing, sentiment analysis, and backtesting.

## Usage

1.  Install dependencies.
2.  Run the notebooks in order to process the data and backtest the strategy.
